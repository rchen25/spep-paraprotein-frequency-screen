spep-paraprotein-frequency-screen REST API Objects & Endpoints

January 22, 2022

This REST API makes paraprotein predictions from densitometric scans of serum protein electrophoresis (SPEP) gels. The method requires the densitometric scan in Sebia's hexadecimal format, such as that produced by a Sebia HYDRASYS instrument. The method uses spatial frequency domain features of the densitometric scan and a Python scikit-learn model that we trained using cases in our clinical immunology laboratory. In the future, we hope to add additional endpoints to support formats beyonds Sebia's hexadecimal format and that will allow REST API clients to train their own Python scikit-learn models.

Demonstrations of the REST API endpoints are available at the links below.

GET /spep/resources/samples Endpoint

A convenience method to instantiante a Sample object.

POST /spep/resources/samples Endpoint

This endpoint makes a paraprotein prediction (i.e., paraprotein detected or paraprotein not detected) given a sample and control curve in a Sample object.
